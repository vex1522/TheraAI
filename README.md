# TheraAI
An intelligent AI-powered mental health assistant that detects emotions from user text and generates empathetic, therapeutic responses using NLP and deep learning techniques.

**Overview** 

AI Therapist is a conversational AI system designed to provide supportive and empathetic responses based on the emotional state of the user. The system first performs emotion classification on the input text and then generates a context-aware therapeutic response to help users feel heard and supported.

The project combines emotion detection models, natural language processing, and dialogue generation techniques to simulate a basic AI counseling assistant.

**Key Features**

1. Emotion Detection

Classifies user emotions such as joy, sadness, anger, fear, surprise, and neutral.

Built using transformer-based NLP models.

2. Empathetic Response Generation

Generates supportive and context-aware responses tailored to the detected emotion.

Designed to mimic therapeutic conversation patterns.

3. Emotion Classification Pipeline

Preprocessing → Tokenization → Model Inference → Emotion Prediction.

4. Efficient NLP Models

Uses transformer architectures and optimized inference for fast response generation.
